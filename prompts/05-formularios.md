# GRID BLOCK
NO modelar Grid como metadato.
Grid es un bloque especializado.
Debe soportar:
* Columnas tipadas
* Fórmulas por fila
* Fórmulas por columna
* Totales
* Agregaciones
* Validaciones
* Límite configurable
* Ilimitado configurable
* Virtual Scrolling
* Paginación eficiente
Analizar alternativas de rendimiento.
